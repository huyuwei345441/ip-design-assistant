### **流程可用性矩阵**

| **流程/场景** | **是否可用IP** | **IP类型** | **使用规范** | **备注** |
| :--- | :--- | :--- | :--- | :--- |
| **启动/闪屏** | **✅ 可用** | 静态/动画IP | 品牌氛围营造 | 建议使用品牌主IP |
| **首页/首页缺省** | **✅ 可用** | 全页空状态IP | 200×200px |  |
| **弹窗（确认/提示）** | **✅ 可用** | 弹窗IP | 120×120px，不挡按钮 |  |
| **活动/营销弹窗** | **✅ 可用** | 大尺寸弹窗IP | 160×160px | 需与活动主题协调 |
| **加载/等待** | **✅ 可用** | 动画IP | 可循环动画，≤2秒 |  |
| **结果页（成功/失败）** | **✅ 可用** | 结果页IP | 区分正向/负向背景 | 关键场景，必须区分 |
| **空状态（无数据/无内容）** | **✅ 可用** | 空状态IP | 按场景选择尺寸 |  |
| **浮标/悬浮入口** | **⚠️ 条件可用** | 小尺寸IP | 不遮挡主要内容 | 需产品经理确认 |
| **下拉刷新** | **⚠️ 条件可用** | 动画IP | 轻量动画，不干扰刷新 |  |
| **支付流程** | **🚫 不可用** | - | - | 强交易场景，避免干扰 |
| **密码/验证码输入** | **🚫 不可用** | - | - | 安全敏感场景 |
| **人脸/生物识别** | **🚫 不可用** | - | - | 安全敏感场景 |
| **协议/合同签署** | **🚫 不可用** | - | - | 合规披露页面 |
| **风险提示/披露** | **🚫 不可用** | - | - | 严肃场景，禁止趣味化 |
| **500/系统错误** | **🚫 不可用** | - | - | 使用系统标准错误页 |
| **数据报表/详情页** | **🚫 不可用** | - | - | 信息密集型页面 |
