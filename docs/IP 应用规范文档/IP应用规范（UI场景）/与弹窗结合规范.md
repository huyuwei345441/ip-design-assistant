弹窗是强引导场景，IP的加入旨在弱化打断感，增强趣味性。但需严格遵循下方的核心设计规范，在视觉表现与交互体验之间取得最佳平衡。

![图片](https://lantaipt016.58wos.com.cn/uVARqPoNmuyu/lantaisystemfile/a22c85dfa6554002a743b07fcdf75c03.png)

- **比例控制原则**

IP形象可适度超出弹窗上沿进行“破形”排列以增强层次感，但建议超出高度严格控制在IP自身高度的**1/3**以内，以维持整体视觉的重心稳定。

- **交互避让红线**

IP形象的任何部分**绝对不可遮挡**关键控件，特别是**关闭按钮 (X)**及**主操作按钮**。必须预留充足的交互热区，确保用户操作路径畅通无阻。

根据弹窗类型，采用不同的IP组合模式，实现视觉与功能的最佳平衡

### **弹窗IP位置规范**

| <mark>**弹窗类型**</mark> | <mark>**IP位置**</mark> | <mark>**IP尺寸**</mark> | <mark>**组合模式**</mark> |
| :--- | :--- | :--- | :--- |
| 确认弹窗 | 弹窗上方居右或居中，可破形 | 高度弹性区间：200-400px | IP + 标题 + 说明 + 按钮 |
| 提示弹窗 | 弹窗上方居右或居中，可破形 | 高度弹性区间：200-400px | IP + 标题 + 说明 + 按钮 |
| 活动/营销弹窗 | 弹窗上方，可破形 | 高度弹性区间：200-400px | IP + 标题 + 内容 + 按钮 |
| 轻提示Toast | 容器内左侧或内右侧 | 高度不小于100px | IP + 文案 |
| 底部弹窗 | 弹窗上方居右或居中，可破形 | 高度弹性区间：200-400px | IP + 标题 + 列表 + 按钮 |

**核心约束：**

- IP上沿可超出弹窗 ≤ 自身高度1/3

- IP下沿距离标题文字 ≥ 32px

- 关闭按钮热区范围：88×88px，IP不得进入此区域

- 主操作按钮上方：IP不得遮挡按钮及其上方16px区域
