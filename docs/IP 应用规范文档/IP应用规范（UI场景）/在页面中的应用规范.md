**核心约束：**

- P形象不得遮挡关键信息文本、按钮、导航元素

- IP色彩需与页面整体色调协调，不可过于跳脱

- 同一页面内IP出现次数不超过1处（浮标除外）

- IP动画时长不超过5秒，避免持续分散注意力
