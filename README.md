# 福宝 IP 设计助手

一个专业的 IP 设计辅助工具，帮助设计师快速创建和管理 IP 形象。

## 功能特性

- **规范顾问** - 专业解读 IP 设计规范
- **AI生图提示词** - 智能生成创意提示词
- **创意工坊** - 探索多样化 IP 创意方案
- **素材库** - 海量 IP 素材资源
- **IP规范手册** - 查看完整的 IP 设计规范

## 技术栈

- HTML5 / CSS3 / JavaScript
- 响应式设计
- 现代化 UI 风格

## 本地运行

```bash
cd public
python3 -m http.server 8080
```

访问 http://localhost:8080
